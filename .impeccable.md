## Design Context

### Users
Travel agency teams (trip managers, coordinators) building and managing luxury/premium FAM trip itineraries. Used daily in a professional desktop context — not a consumer app. Users need to move fast and manage complex multi-day schedules across 8+ simultaneous trips.

### Brand Personality
**Precise. Editorial. Uncompromising.** — "sports car instrument cluster meets luxury magazine layout." Not a friendly consumer app. Not cosy. Urgency, forward motion, and control.

### Aesthetic Direction
- **Dark mode primary**: Void black (`#050505`) backgrounds, `#111111` card surfaces
- **Single accent**: Cyber teal `#0BD2B5` — only for active states, focus rings, and primary CTAs. Never diluted.
- **Type**: Barlow (body/UI) + Barlow Condensed (display/hero labels). All labels, nav, headings → uppercase. Display text → uppercase italic with tight tracking.
- **Anti-references**: generic SaaS dashboards, pastel travel apps, friendly/rounded consumer UIs, gradient-heavy "AI aesthetics"

### Design Principles
1. **Restraint on accent** — `#0BD2B5` only where it earns its place. One accent, used sparingly.
2. **Shadow + border always paired** on dark surfaces — shadow alone has no definition on void black.
3. **Uppercase is structural** — labels, nav, headings always uppercase. Body copy sentence-case only.
4. **Crisp interactions** — 200ms transitions, spring entrances, no bounce easing.
5. **Editorial density** — high information density is a feature, not a problem. Don't pad for whitespace's sake.
